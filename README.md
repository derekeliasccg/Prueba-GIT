# Proyecto de Grado: Sistema de Gestión (POO)
Este proyecto es parte de mi carrera en **Ingeniería en Ciencia de Datos e IA**.

## Descripción del Proyecto
Originalmente desarrollado en **Java**, este sistema ha sido migrado a **Python** para el segundo parcial de la materia Programación Orientada a Objetos.

## Funcionalidades Implementadas
* **Migración Completa:** Traducción de lógica de clases, herencia y polimorfismo de Java a Python.
* **Persistencia de Datos:** Uso de **Serialización** en Python para guardar y cargar el estado de los objetos (jugadores, partidas o datos del sistema).
* **Control de Versiones:** Gestión del ciclo de vida del software mediante Git y GitHub.

## Tecnologías Utilizadas
* **Lenguaje:** Python 3.10+
* **Entorno:** Visual Studio Code
* **Control de Versiones:** Git / GitHub
